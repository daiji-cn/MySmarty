**示例一：调用php内置函数**

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
{time}
</body>
</html>
```

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
{date 'Y-m-d'}
</body>
</html>
```

你也可以自己定义函数，只需要调用即可，参数用空格分割传递。
